# Поиск экскурсий #
Приложение для поиска экскурсий по названию и городу с использованием публичного API Sputnik8.

[Посмотреть демо](https://vue3-tour-search.netlify.app/)


## Функциональность ##
- Поиск экскурсий по названию с поддержкой нечёткого поиска (Fuse.js)

- Фильтрация по городу с кастомным выпадающим списком

- Автодополнение при вводе названия экскурсии

- Интеллектуальный фильтр городов - показываются только города, в которых есть экскурсии

- Адаптивная вёрстка - корректно отображается на всех устройствах

- Состояния загрузки с индикатором (лоадер)

- Обработка ошибок при загрузке данных

- Сброс фильтров одной кнопкой

- Карточки экскурсий с обложкой, рейтингом, ценой и описанием

## Технологии ##

- **Vue 3** (Composition API)

- **TypeScript** - строгая типизация

- **Vite** - быстрая сборка и разработка

- **Fuse.js** - нечёткий поиск

- **@trevoreyre/autocomplete-vue** - автодополнение

- **Fetch API** - запросы к серверу

- **CSS Grid / Flexbox** - адаптивная вёрстка

```bash

git clone https://github.com/Alexandra-Vikhareva/vue3-tour-search.git

# Перейти в папку проекта
cd vue3-tour-search

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

```

## Настройка прокси для API ##
Для обхода CORS при локальной разработке в файле vite.config.ts настроен прокси:
```
ts
server: {
  proxy: {
    '/api': {
      target: 'https://api.sputnik8.com/v1',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api/, '')
    }
  }
}
```
Все запросы к API идут через /api/..., что позволяет избежать CORS-ошибок.
